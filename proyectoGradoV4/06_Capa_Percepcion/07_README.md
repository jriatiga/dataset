## Uso dentro del Framework

Esta capa representa la dimensión social del Framework V7 mediante el uso
del IRCA como proxy de percepción ciudadana. Su finalidad es complementar
las capas climáticas, hidráulicas y de calidad del agua para fortalecer los
modelos predictivos y los análisis integrados de la gestión hídrica.
